# Dromtek

Dromtek adalah komunitas pengembang perangkat lunak dan website mengulas tentang 💻 pemrograman, ⚗️ iptek dan 🛋️ filosofi.

- 🚀 Misi: menjadi ruang aman untuk belajar komputer yang inklusif sekaligus tempat pengembangan diri untuk membangun sesama manusia.

- 🌐 Website: [dromtek](https://dromtek.vercel.app)
